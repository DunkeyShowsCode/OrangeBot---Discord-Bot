 ## OrangeBot -> Discord Bot

<h2>Description</h2>
This is a discord bot that can talk to people who use the proper Command and it also tracks who ever edits their message.
<br />


<h2>Languages and Utilities Used</h2>

- <b>Python 3.11 </b> 
- <b>Discord Import</b>
- <b>Random Import</b>

<h2>Environments Used </h2>

- <b>Pycharm</b>
